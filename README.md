# To Do List
## The Odin Project

[![My Skills](https://skillicons.dev/icons?i=js,html,css,webpack)](https://skillicons.dev)



## Features
* Add tasks with due date and priority to an specific project
* See details of a task
* Add projects
* Remove tasks
